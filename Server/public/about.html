Liftoff Competition Platform

An automated competition system for the Liftoff FPV Simulator. It turns casual multiplayer lobbies into structured, scored racing events with real-time leaderboards, automated track rotation, and weekly competitions.

How It Works

1

Join the Lobby

Connect to the Liftoff multiplayer lobby running the competition plugin. Just fly — the system tracks everything automatically.

2

Race on Curated Tracks

Tracks rotate automatically via managed playlists. Each track runs for a set time before the system advances to the next one.

3

Earn Points

Every race during a competition week is scored. You earn points from race positions, lap counts, fastest laps, consistency, and more.

4

Climb the Leaderboard

Weekly scores are aggregated into a season leaderboard. Compete across multiple weeks to claim the top spot.

Competition Scoring

Position Points

F1-style scoring based on your finishing position each race.

1st: 25 · 2nd: 18 · 3rd: 15 · 4th: 12 · 5th: 10 · 6th: 8 · 7th: 6 · 8th: 4

Lap Points

Rewarded for putting in the laps. Every 5 completed laps earns a bonus point.

Hot Streak

Set the fastest lap of the race and earn bonus points. Rewards raw speed.

Consistency

Points for maintaining tight lap times. The smaller the gap between your best and worst laps, the higher you score.

Improvement

Rewarded for getting faster throughout the race. Show progression and you earn extra points.

Participation

Show up, race, and earn points just for being part of the competition. More races = more points.

In-Game Chat Commands

Command Description
/info Shows available commands and time remaining on the current track
/next Vote to skip the current track. Starts a 3-minute vote — needs a majority to pass
/extend Vote to extend the current track by 5 minutes. Also needs a majority vote
/stay Reset your idle timer if you're about to be kicked for inactivity

Idle Kick System

When the lobby is full (8 players), an idle detection system activates to free up slots for active pilots. If you haven't completed a lap, checkpoint, or sent a chat message for 5 minutes, you'll receive a warning. After another minute of inactivity, you'll be removed from the lobby.

Use the /stay command to reset your idle timer if you need a break but want to keep your spot.

How the System Works

The platform consists of three main components working together:

Game Plugin — A BepInEx mod running inside the Liftoff simulator. It captures race events (laps, positions, player joins/leaves) and sends them to the server over a WebSocket connection. It also receives commands from the server to change tracks, send chat messages, and kick players.

Server — A Node.js backend that processes all race events, manages competitions, runs scoring, handles track playlists, and serves the public website. Data is stored in SQLite. Real-time updates are pushed to spectators via WebSocket.

Public Website — The pages you're browsing now. Live race data, competition standings, and historical stats — all updated in real time.
